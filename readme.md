#Kirby Bundle

Kirby CMS snippets for Sublime Text 2.

##Snippets
Snippets are based on kirby [cheat sheet](http://getkirby.com/content/02-docs/kirby-cheatsheet.pdf). so "first()" will expand into:
   
		$site->uri()->path()->first()
